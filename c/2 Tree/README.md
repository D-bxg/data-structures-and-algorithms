># 满二叉树
>## 高度为h时有$2^h-1$个结点
>## 结点i的左孩子为2i，右孩子为2i+1
>## 父结点为向下取整的i/2

># 完全二叉树
>## 不满的满二叉树称作完全二叉树
>## 满二叉树是满了的完全二叉树
>## 完全二叉树最多有一个度为一的结点

># 二叉排序树
>## 左子树上的结点的关键字都小于根节点
>## 右子树上的结点的关键字都大于根节点
>## 子树也是二叉排序树
>## 从最左到最右逐渐增加
>## 用于排序搜索

># 平衡二叉树
>## 任意结点的左子树和右子树深度之差不超过1（小于等于）
>## 更高的搜索效率
